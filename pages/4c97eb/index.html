Viteзҡ„дҪҝз”Ё

# Viteзҡ„дҪҝз”Ё

# и®ӨиҜҶVite

Viteзҡ„е®ҡдҪҚжҳҜдёӢдёҖд»ЈеүҚз«ҜејҖеҸ‘дёҺжһ„е»әе·Ҙе…·пјҢиғҪжҳҫи‘—жҸҗеҚҮеүҚз«ҜејҖеҸ‘дҪ“йӘҢгҖӮ

е®һйҷ…ејҖеҸ‘дёӯпјҢзј–еҶҷзҡ„д»Јз ҒеҫҖеҫҖдёҚиғҪиў«жөҸи§ҲеҷЁзӣҙжҺҘиҜҶеҲ«пјҢеҰӮES6гҖҒTypeScriptгҖҒVueзӯүгҖӮ

жүҖд»Ҙеҝ…йЎ»йҖҡиҝҮжһ„е»әе·Ҙе…·еҜ№д»Јз ҒиҝӣиЎҢиҪ¬жҚўгҖҒзј–иҜ‘пјҢеҰӮ webpack (opens new window)гҖҒRollup (opens new window) е’Ң Parcel (opens new window) зӯүе·Ҙе…·пјҢе®ғд»¬жһҒеӨ§ең°ж”№е–„дәҶеүҚз«ҜејҖеҸ‘иҖ…зҡ„ејҖеҸ‘дҪ“йӘҢгҖӮ

дҪҶжҳҜйҡҸзқҖйЎ№зӣ®и¶ҠжқҘи¶ҠеӨ§пјҢйңҖиҰҒеӨ„зҗҶзҡ„JavaScriptд»Јз ҒйҮҸе‘ҲжҢҮж•°зә§еўһй•ҝгҖӮ

жһ„е»әе·Ҙе…·йңҖиҰҒеҫҲй•ҝж—¶й—ҙжүҚиғҪејҖеҗҜжңҚеҠЎеҷЁпјҢHMRд№ҹйңҖиҰҒеҮ з§’й’ҹжүҚиғҪеңЁжөҸи§ҲеҷЁдёӯеҸҚжҳ еҮәжқҘпјҢејҖеҸ‘дҪ“йӘҢеҫҲе·®гҖӮ

Vite ж—ЁеңЁеҲ©з”Ёз”ҹжҖҒзі»з»ҹдёӯзҡ„ж–°иҝӣеұ•и§ЈеҶідёҠиҝ°й—®йўҳпјҡжөҸи§ҲеҷЁејҖе§ӢеҺҹз”ҹж”ҜжҢҒ ES жЁЎеқ—пјҢдё”и¶ҠжқҘи¶ҠеӨҡ JavaScript е·Ҙе…·дҪҝз”Ёзј–иҜ‘еһӢиҜӯиЁҖзј–еҶҷгҖӮ

  • е®ғдё»иҰҒз”ұдёӨйғЁеҲҶз»„жҲҗпјҡ
    • дёҖдёӘејҖеҸ‘жңҚеҠЎеҷЁпјҢе®ғеҹәдәҺ еҺҹз”ҹ ES жЁЎеқ— (opens new window) жҸҗдҫӣдәҶ дё°еҜҢзҡ„еҶ…е»әеҠҹиғҪ (opens new window)пјҢеҰӮйҖҹеәҰеҝ«еҲ°жғҠдәәзҡ„ жЁЎеқ—зғӯжӣҙж–°пјҲHMRпјү (opens new window)гҖӮ
    • дёҖеҘ—жһ„е»әжҢҮд»ӨпјҢе®ғдҪҝз”Ё Rollup (opens new window) жү“еҢ…дҪ зҡ„д»Јз ҒпјҢе№¶дё”е®ғжҳҜйў„й…ҚзҪ®зҡ„пјҢеҸҜиҫ“еҮәз”ЁдәҺз”ҹдә§зҺҜеўғзҡ„й«ҳеәҰдјҳеҢ–иҝҮзҡ„йқҷжҖҒиө„жәҗгҖӮ

# жөҸи§ҲеҷЁеҺҹз”ҹжЁЎеқ—еҢ–ж”ҜжҢҒ

еҰӮжһңдёҚеҖҹеҠ©е…¶е®ғе·Ҙе…·пјҢеҠ иҪҪдёҠзҷҫдёӘжЁЎеқ—зҡ„JSд»Јз ҒпјҢеҜ№дәҺжөҸи§ҲеҷЁиҜ·жұӮжҳҜе·ЁеӨ§зҡ„ж¶ҲиҖ—гҖӮеҰӮеҠ иҪҪLoadashпјҢеҸҜиғҪжҲ‘д»¬еҸӘдҪҝз”Ёе…¶дёӯеҮ дёӘж–№жі•пјҢдҪҶжҳҜжөҸи§ҲеҷЁйңҖиҰҒеҠ иҪҪеҮ зҷҫдёӘж–Үд»¶пјҢж¶ҲиҖ—е·ЁеӨ§гҖӮ

е…¶ж¬ЎпјҢд»Јз ҒеҸҜиғҪжҳҜTypeScriptгҖҒlessгҖҒvueзӯүпјҢжөҸи§ҲеҷЁдёҚиғҪзӣҙжҺҘиҜҶеҲ«гҖӮ

Viteи§ЈеҶідәҶдёҠйқўзҡ„иҝҷдәӣй—®йўҳгҖӮ

д»Јз ҒеҰӮдёӢпјҡ

// index.js
import { formatDate } from './utils/format.js'
import _ from '../node_modules/lodash-es/lodash.default.js';


console.log('Hello Vite');

console.log(formatDate());

console.log(_.join(['A', 'B', 'C']))

// format.js
export function formatDate() {
  return '2022-07-03';
}

# Viteзҡ„дҪҝз”Ё

йҖҡиҝҮдёӢйқўе‘Ҫд»Өе®үиЈ…ViteпјҢдҫҝдәҺеӯҰд№ жһ„е»әе·Ҙе…·гҖӮ

npm i -D vite

жіЁпјҡе®ҳж–№ж–ҮжЎЈдёӯйҖҡиҝҮnpm create vite@latestжҳҜе°ҶViteдҪңдёәи„ҡжүӢжһ¶жқҘеҲӣе»әйЎ№зӣ®пјҢдёҚж–№дҫҝеҫҲеӨҡз»ҶиҠӮзҡ„зҗҶи§ЈгҖӮ

жӯӨж—¶дҝ®ж”№lodashеә“зҡ„еј•е…Ҙж–№ејҸпјҢжүӢеҠЁеј•е…ҘViteдёҚдјҡеҒҡд»»дҪ•зҡ„дјҳеҢ–пјҡ

import { formatDate } from './utils/format.js'
// import _ from '../node_modules/lodash-es/lodash.default.js';
import _ from 'lodash-es';


console.log('Hello Vite');

console.log(formatDate());

console.log(_.join(['A', 'B', 'C']))

npx viteжқҘеҗҜеҠЁйЎ№зӣ®пјҢжӯӨж—¶еҸҜд»ҘзңӢеҲ°пјҢNetworkдёӯеҸӘе…ідәҺLodashеҸӘеҠ иҪҪдәҶlodash-es.jsж–Үд»¶гҖӮ

Viteе°ҶиҝҷдёӘLodashеә“еҒҡдәҶеҗҲе№¶пјҢеҮҸе°‘HTTPзҡ„иҜ·жұӮж¬Ўж•°пјҢеңЁжү“еҢ…йҳ¶ж®өеҶҚйҖҡиҝҮTreeShakingиҝӣиЎҢиҝӣдёҖжӯҘдјҳеҢ–гҖӮ

# Viteзҡ„еҠҹиғҪ

https://cn.vitejs.dev/guide/features.html

еҜ№йқһеёёеҹәзЎҖзҡ„дҪҝз”ЁжқҘиҜҙпјҢдҪҝз”Ё Vite ејҖеҸ‘е’ҢдҪҝз”ЁдёҖдёӘйқҷжҖҒж–Үд»¶жңҚеҠЎеҷЁе№¶жІЎжңүеӨӘеӨ§еҢәеҲ«гҖӮз„¶иҖҢпјҢVite иҝҳйҖҡиҝҮеҺҹз”ҹ ESM еҜје…ҘжҸҗдҫӣдәҶи®ёеӨҡдё»иҰҒз”ЁдәҺжү“еҢ…еңәжҷҜзҡ„еўһејәеҠҹиғҪгҖӮ

ж–ҮжЎЈдёӯеҜ№еҗ„еҠҹиғҪеҒҡдәҶиҜҰз»Ҷд»Ӣз»ҚпјҢиҝҷйҮҢеҜ№ејҖеҸ‘дёӯж”ҜжҢҒзҡ„еёёи§Ғж–Үд»¶зұ»еһӢеҒҡжј”з»ғгҖӮ

# HMR

Vite жҸҗдҫӣдәҶдёҖеҘ—еҺҹз”ҹ ESM зҡ„ HMR API (opens new window)гҖӮ е…·жңү HMR еҠҹиғҪзҡ„жЎҶжһ¶еҸҜд»ҘеҲ©з”ЁиҜҘ API жҸҗдҫӣеҚіж—¶гҖҒеҮҶзЎ®зҡ„жӣҙж–°пјҢиҖҢж— йңҖйҮҚж–°еҠ иҪҪйЎөйқўжҲ–жё…йҷӨеә”з”ЁзЁӢеәҸзҠ¶жҖҒгҖӮ

# ж–Үд»¶еӨ„зҗҶз®Җжһҗ

еҸҜд»ҘзңӢеҲ°пјҢViteеңЁиҝ”еӣһж–Үд»¶ж—¶пјҢзӣҙжҺҘиҝ”еӣһдәҶеҺҹж–Үд»¶зұ»еһӢпјҢеҰӮtsгҖҒlessгҖҒscssпјҢдҪҶиҝҷдәӣзұ»еһӢжөҸи§ҲеҷЁжң¬иә«жҳҜдёҚж”ҜжҢҒзҡ„гҖӮ

йӮЈд№Ҳдёәд»Җд№ҲжөҸи§ҲеҷЁзӣҙжҺҘиҝ”еӣһиҝҷдәӣзұ»еһӢе‘ўпјҹ

еҺҹеӣ жҳҜпјҢViteжҸҗеүҚе°ҶжүҖжңүж јејҸзҡ„ж–Үд»¶пјҢжҸҗеүҚзј–иҜ‘жҲҗдәҶJSж–Үд»¶пјҢз„¶еҗҺViteдёӯзҡ„connectжңҚеҠЎеҷЁпјҢйҖҡиҝҮHTTP 304йҮҚе®ҡеҗ‘еҲ°е·Із»Ҹзј–иҜ‘еҘҪзҡ„JSж–Үд»¶гҖӮиҺ·еҸ–еҲ°зј–иҜ‘еҘҪзҡ„ж–Үд»¶еҗҺпјҢжөҸи§ҲеҷЁеҶҚиҝӣиЎҢи§ЈжһҗгҖӮ

еҸҜд»ҘзңӢеҲ°пјҢжүҖжңүзҡ„ж–Үд»¶йғҪжҳҜJSж–Үд»¶пјҢе№¶дёҚжҳҜеҺҹж–Үд»¶гҖӮ

йҖҡиҝҮйҮҚе®ҡеҗ‘пјҢе°ҶиҜ·жұӮдҝ®ж”№дёәеҸҰдёҖдёӘиҜ·жұӮпјҢconnectж“…й•ҝеӨ„зҗҶиҝҷж ·зҡ„д»»еҠЎгҖӮ

# Viteдёӯзҡ„connect

Vite2еҗҺпјҢViteеҶ…йғЁдҪҝз”Ёconnect (opens new window)еә“жқҘеҒҡжңҚеҠЎгҖӮе·Із»ҸдёҚеҶҚдҪҝз”ЁKoaдәҶпјҲVite1дҪҝз”Ёзҡ„жҳҜKoaпјүпјҢиҮідәҺдёәд»Җд№ҲжҚўжҲҗConnectпјҢе®ҳж–№ж–ҮжЎЈдёӯеҒҡдәҶзӣёе…іиҜҙжҳҺ

Since most of the logic should be done via plugin hooks instead of middlewares, the need for middlewares is greatly reduced. The internal server app is now a good old connect (opens new window) instance instead of Koa.

з”ұдәҺеӨ§еӨҡж•°йҖ»иҫ‘еә”иҜҘйҖҡиҝҮжҸ’д»¶й’©еӯҗиҖҢдёҚжҳҜдёӯй—ҙд»¶жқҘе®ҢжҲҗпјҢеӣ жӯӨеҜ№дёӯй—ҙд»¶зҡ„йңҖжұӮеӨ§еӨ§еҮҸе°‘гҖӮеҶ…йғЁжңҚеҠЎеҷЁеә”з”ЁзҺ°еңЁжҳҜдёҖдёӘеҫҲеҘҪзҡ„ж—§зүҲзҡ„connectе®һдҫӢпјҢиҖҢдёҚжҳҜKoaгҖӮ

# Viteдёӯзҡ„ESBuild

Vite дҪҝз”Ё esbuild (opens new window) йў„жһ„е»әдҫқиө– (opens new window)гҖӮesbuild дҪҝз”Ё Go зј–еҶҷпјҢе№¶дё”жҜ”д»Ҙ JavaScript зј–еҶҷзҡ„жү“еҢ…еҷЁйў„жһ„е»әдҫқиө–еҝ« 10-100 еҖҚгҖӮ

esbuildзҡ„зү№зӮ№пјҡ

  • и¶…еҝ«зҡ„жһ„е»әйҖҹеәҰпјҢдё”дёҚйңҖиҰҒзј“еӯҳ
  • ж”ҜжҢҒES6е’ҢCommonJSжЁЎеқ—еҢ–
  • ж”ҜжҢҒES6зҡ„Tree Shaking
  • ж”ҜжҢҒGoгҖҒJavaScriptзҡ„API (opens new window)
  • ж”ҜжҢҒTypeScript (opens new window)е’Ң JSX (opens new window) иҜӯжі•зј–иҜ‘
  • ж”ҜжҢҒSourceMap (opens new window)
  • ж”ҜжҢҒд»Јз ҒеҺӢзј© (opens new window)
  • ж”ҜжҢҒжү©еұ•е…¶е®ғжҸ’д»¶ (opens new window)

жңүдәҶesbuildпјҢе°ұдёҚйңҖиҰҒеҶҚдҪҝз”ЁBabelеҜ№JSд»Јз ҒеҒҡеӨ„зҗҶгҖӮ

esbuildдёәд»Җд№Ҳиҝҷд№Ҳеҝ«пјҹ

  • esbuildдҪҝз”ЁGoиҜӯиЁҖзј–еҶҷпјҢеҸҜд»ҘзӣҙжҺҘиҪ¬жҚўжҲҗжңәеҷЁз ҒпјҢиҖҢж— йңҖз»ҸиҝҮеӯ—иҠӮз ҒгҖӮJavaScriptд»Јз ҒйңҖиҰҒе…Ҳзј–иҜ‘и§ЈжһҗжҲҗAST вҖ”> иҪ¬жҲҗеӯ—иҠӮз Ғ вҖ”> дјҳеҢ–еӯ—иҠӮз Ғ вҖ”> иҪ¬жҲҗжңәеҷЁз ҒгҖӮGoжҜ”JavaScriptеҝ«еҫҲеӨҡгҖӮ
  • esbuildеҸҜд»Ҙе……еҲҶеҲ©з”ЁCPUзҡ„еӨҡеҶ…ж ёпјҢesbuildдјҡеҚ•зӢ¬ејҖдёҖдёӘиҝӣзЁӢпјҢеңЁиҝҷдёӘиҝӣзЁӢдёӯе®ғдјҡе°ҪеҸҜиғҪеӨҡең°ејҖеҗҜзәҝзЁӢпјҢеӨҡзәҝзЁӢиҝҗиЎҢеңЁCPUеӨҡдёӘеҶ…ж ёдёӯпјҢе№¶иЎҢжү§иЎҢпјҢж•ҲзҺҮжӣҙй«ҳпјҢе№¶дё”е°ҪеҸҜиғҪи®©CPUйҘұе’ҢиҝҗиЎҢгҖӮ
  • esbuildзҡ„жүҖжңүеҶ…е®№йғҪжҳҜд»Һйӣ¶ејҖе§Ӣзј–еҶҷзҡ„пјҢиҖҢдёҚжҳҜдҪҝз”Ёз¬¬дёүж–№пјҢжүҖд»Ҙд»ҺдёҖејҖе§Ӣе°ұиҖғиҷ‘еҲ°дәҶеҗ„з§ҚжҖ§иғҪй—®йўҳгҖӮ
